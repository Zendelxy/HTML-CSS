設定/製作動畫

@keyframes name {...} === @keyframes "name" {...} ※name等效於"name"，名稱之大小寫有區別，特殊名稱(如none)加上" "後可用
製作動畫時，需先定義關鍵幀(keyframe)，如開頭、結尾和過程，開頭可用from或0%表示，結尾可用to或100%表示，過程則只能以百分比(n%)表示 @keyframes name { from{...} n%{...} to{...} }

animation-name動畫名稱

指定所要執行的動畫名稱

animation-duration

動畫持續時間(s/ms)

animation-delay

動畫執行前的延遲時間(s/ms)

animation-timing-function

此屬性定義過渡時的速度曲線，可用之值同transition-timing-function

animation-direction

動畫播放方向
  • normal:預設值
  • reverse:反轉播放
  • alternate:正反輪流播放(奇數正，偶數反)
  • alternate-reverse:與alternate相反(奇數反，偶數正)

animation-fill-mode動畫執行之外的樣式

  • none:預設值，結束時返回原始狀態
  • forwards:結束時保持在最後的影格
  • backwards:未開始時保持在第一個影格，結束時保持在第一個影格
  • both:未開始時保持在第一個影格，結束時保持在最後的影格
做個表格以方便理解:
值 開始前(delay) 結束
none 原始模樣 原始模樣
forwards 原始模樣 100%時的模樣
backwards 0%時的模樣 原始模樣
both 0%時的模樣 100%時的模樣

animation-play-state播放狀態

  • running:預設值，運行
  • paused:暫停

animation-iteraion-count播放次數

  • n:預設為1，可自訂次數
  • infinite:無限次

animation複合屬性

同transition，除了duration第一、delay第二以外沒有順序限制
※animation-play-state一般單獨使用

transition與animation差異

啟動:transition啟動像「手排」，需要觸發條件才會動。animation啟動則像「自排」，會自動開始，不過也能以JS控制
操縱:transition只能從狀態A到狀態B。animation則可以設定關鍵幀
