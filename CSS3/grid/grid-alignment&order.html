justify-* grid水平對齊

justify-self 單一item在格子內的水平對齊

  • stretch:預設值，拉伸填滿格子寬
  • auto:justify-items的設定
  • start:對齊起點，以容器書寫方向為主，rtl為右，ltr為左
  • end:對齊終點，以容器書寫方向為主，rtl為左，ltr為右
  • center:置中
  • left:靠左，不受文字方向影響
  • right:靠右，不受文字方向影響
  • self-start:以item書寫方向為主，rtl為右，ltr為左
  • self-end:以item書寫方向為主，rtl為左，ltr為右
※在現在大多使用單一語言的情境下，left/right/self-start/self-end與start/end效果是一樣的

justify-items 所有item在格子內的水平對齊

  • stretch:預設值，拉伸填滿格子寬
  • start:對齊起點，以容器書寫方向為主，rtl為右，ltr為左
  • end:對齊終點，以容器書寫方向為主，rtl為左，ltr為右
  • center:置中
  • left:靠左，不受文字方向影響
  • right:靠右，不受文字方向影響
  • self-start:以item書寫方向為主，rtl為右，ltr為左
  • self-end:以item書寫方向為主，rtl為左，ltr為右

justify-content 每排item在row中的水平對齊

與前兩者的差別在於，對齊並非是在格子中而是容器中
  • stretch:預設值，拉伸格子填滿寬
  • start:對齊起點，以容器書寫方向為主，rtl為右，ltr為左
  • end:對齊終點，以容器書寫方向為主，rtl為左，ltr為右
  • center:置中
  • space-between:左右靠邊，均分間距
  • space-around:均分間距，所有元素左右間距相同
  • space-evenly:包括起點、終點一起均分間距

align-* grid垂直對齊

align-self 單一item在格子內的垂直對齊

  • stretch:預設值，拉伸填滿格子高
  • auto:繼承align-items的設定
  • start:對齊起點
  • end:對齊終點
  • center:置中
  • self-start:以item書寫方向為主，主要影響直式書寫
  • self-end:以item書寫方向為主，主要影響直式書寫

align-items 所有item在格子內的垂直對齊

  • stretch:預設值，拉伸填滿格子高
  • start:對齊起點
  • end:對齊終點
  • center:置中
  • self-start:以item書寫方向為主，主要影響直式書寫
  • self-end:以item書寫方向為主，主要影響直式書寫

align-content 每排item在column中的垂直對齊

與前兩者的差別在於，對齊並非是在格子中而是容器中
  • stretch:預設值，拉伸格子填滿高
  • start:對齊起點
  • end:對齊終點
  • center:置中
  • space-between:左右靠邊，均分間距
  • space-around:均分間距，所有元素左右間距相同
  • space-evenly:包括起點、終點一起均分間距

對齊的複合屬性

  • place-self -> align-self + justify-self
  • place-items -> align-items + justify-items
  • place-content -> align-content + justify-content
※只填一值則兩者共用

order item順序

此屬性同flex，預設為0，值越小則離起點越近
※只改變順序，不改變層級(z-index)
