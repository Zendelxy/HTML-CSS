grid布局

又稱格線布局或網格布局，透過在container(display為grid的元素)中畫線的方式分割區域，再將子元素(item)放入
grid同flex一樣可用inline-*作為display值
※grid item也同flex item一樣會block化
grid不支援float,clear,vertical-align,multiple-columns的相關屬性
※對於無標籤元素，會產生一個虛擬標籤包裹住他

grid基礎架構

track
area
cell
  • grid line格線:劃分出各個區域的線
  • grid track軌道:兩條格線之間的空間
  • grid cell單元格:格線畫出的1x1的空間
  • grid area區域:格線畫出的大於等於1x1的空間，且必須為矩形

grid-template-columns/rows grid容器列高/行高

以空白隔開多個數值
尺寸設定方式:
  1. 固定尺寸:%, px, vh/vw...
  2. 彈性尺寸:fr，將剩餘空間按比例分配
  3. 自動適應內容
    • max-content內容所需空間的最大值
    • min-content內容所需空間的最小值
    • minmax(min,max)將寬度限制在min~max間
  4. 限制最大尺寸:fit-content()，可填<length>或<percentage>
    限制寬度不超過指定值，但會至少保證內容正常顯示(min-content)
    完整公式:min(max-content,max(arug,min-content))
  5. 重複產生:repeat(次數,設定)
    次數:
    1. 非負整數
    2. auto-fill
    3. auto-fit
    兩者差異:auto-fill會建立空軌道來觀察剩餘空間，auto-fit則不會
    ※用justify-content:center可以輕易觀察到差別
    auto-fill:
    1
    2
    3
    4
    5
    auto-fit:
    1
    2
    3
    4
    5
  6. auto:根據內容&剩餘空間自動伸縮(min-content~max-content之間)

    7. 格線命名

    方式
    1. grid-template-columns/rows:[名1] px [名3] px [名3]
      在每個track前後加上名稱，要為同一條線設定多個名字須以空格隔開([名1 名2])
    2. grid-template-area自動產生:
      grid-template-areas:"a a"
                    "b .";
      ※"."代表此格為空白格
      ※必須是矩形
      ※自動產生時，track默認為auto
    3. grid-template:grid-template-areas+grid-template-rows/columns
      grid-template:[c] "a a" 10px
              [d] "b ." 20px
              [e]/[f] 30px [g] 40px [h]


      ※格式為先rows再columns，且不支援repeat
