grid-auto-flow 網格流向

決定自動放置的方向&方式
  • row:預設值，水平填滿一列，再換下列
  • column:垂直填滿一行，再換下行
  • row dense:同row，但會回填剩餘空間
  • rcolumn dense:同column，但會回填剩餘空間
※不指定起始格線(grid-row/column-start)時，grid-auto-flow才有用

grid-auto-columns/rows 自動格線

設定自動生成時的軌道大小，複數值以空格隔開
  • <length>
  • fr
  • minmax()
生效時機:
  1. item數超出已定義的格子數
  2. 手動指定超出範圍的位置

grid的兩種縮寫

  1. template: grid-template-rows+grid-template-columns+grid-template-areas(與grid-template縮寫效果一樣)
  2. auto-flow: grid-auto-rows+grid-auto-columns+grid-auto-flow(實驗性，不推薦用)
    語法:
    • row:auto-flow (dense) row / column
    • column:row / auto-flow (dense) column

網格間距

決定欄與列之間的間距，有兩種寫法，屬性名不同但效果相同，建議使用新的語法就好
舊:
  • grid-row-gap 每列間距
  • grid-column-gap 每欄間距
  • grid-gap 同時設定row+column的間距，寫法也是row+column，只有一個值則列欄間距共用
新:
  • row-gap 每列間距
  • column-gap 每欄間距
  • gap 同時設定row+column的間距，寫法也是row+column，只有一個值則列欄間距共用
新寫法已規範成能與flex, multiple-columns等布局通用

item邊界

屬性:
  • grid-row-start item上邊界
  • grid-row-end item下邊界
  • grid-column-start item左邊界
  • grid-column-end item右邊界
值:
  • auto，預設值，對於start表現為依演算法自動尋找，對於end則是跨一行/列(span 1)
  • 自訂行/列
  • span <number>，表示跨n行，start,end皆可用，但另一個需要指定(不能同時用，同時使用會導致兩個值都變auto)
指定方式:
  1. 編號:由1開始，可寫負值，-n代表倒數n條線
  2. 名稱:填入自己命名的格線名
    ※格線同名時，以 名稱 出現順序 的方式來指定，例如:n 1即是第一條叫n的格線
  3. 網格名稱:本質上與上一項相同，以grid-template-areas定義區域後自動生成的格線名(xx-start, xx-end等)來指定

item邊界縮寫

  • grid-row:grid-row-start/grid-row-end
  • grid-column:grid-column-start/grid-column-end
  • grid-area:grid-row-start/grid-column-start/grid-row-end/grid-column-end

grid-area 網格區域

可作為item邊界縮寫，也可以與grid-template-areas配合直接使用區域名稱來放置item
※會無視元素在html中的順序，直接將其放至指定區域
